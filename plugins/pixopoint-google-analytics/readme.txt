== Description ==

The PixoPoint Google Analytics plugin is a cut down uber simple plugin which allows individual site owners to 
simply insert their Analytics code with no other "frills" so they can track their sites performance, as well as 
allowing network owners for multi-site networks to their own Analytics code. Simply enter your tracking code via
Site Admin > Settings > Google Analytics. Individual site owners will be able to enter their own analytics code 
in without interfering with your cross-network tracking code.

= Features =

* Simple to use<br />
* No "branding" for non super-admins<br />
* No frills. Mega simple - ideal for multi-site setups<br />

Install and activate. Pretty simple :)

== Frequently Asked Questions ==

= Does it work for WordPress version x.x.x? =

Probably, but we only intend to actively support the latest version of WordPress.

= Why isn't the plugin in my language? =

The plugin supports localization using PO and MO files. If you provide a translation we are more than happy to include it in future releases. If you've translated the plugin please let us know by posting in our <a href="http://pixopoint.com/forum/">support forum</a>.
